# use of IT

    “It” কী এবং কেন ব্যবহার করা হয়

    “It” English grammar এ খুব important pronoun।
    এটি সাধারণত noun-এর পরিবর্তে বা কোনো general/unknown/impersonal idea বোঝাতে ব্যবহার হয়।

    “It” as a Pronoun (Noun-এর পরিবর্তে)
        যখন আগের sentence এ কোনো noun mention করা থাকে, তখন আবার repeat না করে “it” ব্যবহার করা হয়।

    Example:
        I have a book. It is very interesting.

    এখানে:
    book → it

    “It” for Animals or Objects

    একটা নির্দিষ্ট জিনিস বা প্রাণী বোঝাতে “it” ব্যবহার হয় (যখন gender important না)।

    Example:
        The dog is running. It is fast.

    “It” for Time, Weather, Distance

    এই ক্ষেত্রে “it” কোনো real noun replace করে না, শুধু situation বোঝায়।

    a. Time:
        It is 5 o’clock.

    b. Weather:
        It is raining.

    c. Distance:
        It is 5 kilometers from here.

    “It” as Dummy Subject (Very Important)

    যখন sentence এ real subject পরে আসে, তখন “it” শুধু structure ঠিক করার জন্য ব্যবহার হয়।

    Example:
        It is important to study.

    আসল subject:
        to study

    আরও example:
        It is easy to learn English.

    “It” for General Idea / Situation

    Example:
        It is good to help others.
        It is difficult to solve this problem.

    “It” in Emphasis Structure

    Structure:
        It is/was + noun/pronoun + who/that + verb

    Example:
        It is Rahim who called me.
        It was yesterday that I met him.

    Difference: “It” vs “There”

    It:
        ✔ নির্দিষ্ট কিছু বোঝায়
        ✔ subject হিসেবে কাজ করে
    Example: It is a pen.

    There:
        ✔ অস্তিত্ব বোঝায়
        ✔ dummy subject
    Example: There is a pen.

    Easy Memory Trick
        It = নির্দিষ্ট জিনিস / situation / time / weather
        There = কোনো কিছু আছে বোঝায়

* Practice:
  আমার একটি বই আছে। এটি খুব মজার।
  কুকুরটি দৌড়াচ্ছে। এটি দ্রুত।
  এখন ৫টা বাজে।
  বৃষ্টি হচ্ছে।
  এখান থেকে ৫ কিলোমিটার দূরে।
  পড়া গুরুত্বপূর্ণ।
  ইংরেজি শেখা সহজ।
  অন্যকে সাহায্য করা ভালো।
  এই সমস্যাটি সমাধান করা কঠিন।
  এটি রহিম যে আমাকে ফোন করেছে।
  এটি গতকাল ছিল যে আমি তাকে দেখেছি।
  এটি একটি কলম।

# “It vs There vs This vs That” (Clear Difference)

    এই চারটা word English speaking এ খুব confusing, তাই একবারে সহজ করে বুঝি।

    “It” এর ব্যবহার

    “It” ব্যবহার হয়:
        ✔ আগে বলা noun-এর পরিবর্তে
        ✔ time, weather, distance
        ✔ general idea
        ✔ dummy subject

    Example:
        I bought a phone. It is expensive.
        It is raining.
        It is 10 o’clock.
        It is easy to learn English.

    Simple idea:
        It = নির্দিষ্ট কিছু / situation / idea

    “There” এর ব্যবহার

        “There” ব্যবহার হয়:
            ✔ কোনো কিছু আছে বোঝাতে (existence)

    Example:
        There is a book on the table.
        There are many students.

    Simple idea:
        There = আছে / exist করে

    “This” এর ব্যবহার

        “This” ব্যবহার হয়:
            ✔ কাছের কোনো জিনিস বোঝাতে
            ✔ singular object

    Example:
        This is my pen.
        This book is mine.

    Simple idea:
        This = এই (near + singular)

    “That” এর ব্যবহার

        “That” ব্যবহার হয়:
            ✔ দূরের কোনো জিনিস বোঝাতে
            ✔ singular object

    Example:
        That is my house.
        That boy is my friend.

    Simple idea:
        That = ওই / সেই (far + singular)

    Quick Comparison Table

        It → specific thing / idea
        There → existence (আছে)
        This → near object
        That → far object

    Easy Examples Together
            There is a book on the table.
            It is very useful.
            This is my book.
            That is his book.

    Exam Trick
        ✔ If existence → There
        ✔ If identity/description → It
        ✔ If near → This
        ✔ If far → That
